# 🐾 PITUTI App

Aplicação fullstack para gestão colaborativa de cuidados com pets.

## Stack

- **Frontend:** React + TypeScript + Tailwind CSS v4 + React Router
- **Backend:** Express (Node.js)
- **Dados:** LocalStorage + API REST
